# CNN Multiclass Image Classification (Facial Emotions)

## Task
Multiclass image classification of facial emotions using a CNN.

## Dataset
- 4 emotion classes
- Image size: 96Ã—96
- Manual train / validation / test split (70 / 15 / 15)

## Model
- CNN trained from scratch
- Data augmentation (flip, rotation, zoom)
- Dropout for regularization
- Early stopping

## Results
- Validation accuracy: ~71%
- Test accuracy: ~70%

## Notes
This is a clean baseline CNN. Further improvements are possible with transfer learning.
